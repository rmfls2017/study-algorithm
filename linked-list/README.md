## Two Pointer 관계 정리

```
문제 유형              →  두 포인터의 관계
─────────────────────────────────────────
중간 찾기              →  속도 차이 (2:1)
뒤에서 k번째 찾기       →  간격 유지 (k칸)
사이클 찾기            →  속도 차이 (결국 만남)
```
